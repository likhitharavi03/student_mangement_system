About Student Management System

The Student Management System is a web-based application developed using Flask and MySQL.

It helps manage student records such as personal details, branch, marks, and contact information in an organized way.

Features include:

  • User Authentication (Login & Register)
  • Add, Edit, Delete Students
  • Secure Password Handling
  • Simple and Card-Based UI

This system is ideal for colleges and learning institutions.
